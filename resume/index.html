Web, UX/UI Designer & Frontend Developer

In a general workflow, I am responsible for conceptualizing ideas from sketches to hi-fi wireframes. Before the develop phase, I am required to present the functional specification, and style guide so the functionality and aesthetic aspects of the design are extendable.

I've built prototypes with Axure, Principle or directly with HTML/CSS/JS, then iterate the prototype design in a timely manner.

During developing phase, I would commit my work via git. I use Sass + Gulp to compile SCSS. BEM is the main approach I use for CSSing. As it's easy to reuse the codebase and to keep the CSS specification level low, which is great for maintenance and extends the design if need to.

In some projects, I will use D3.js to visualize data, it's a great and beautiful tool to work with. I have a lot of fun working with it.
